BOS Factory: AI-Enhanced Process Guide

The BOS Factory is powered by a set of GenAI prompts that dramatically simplify the process of creating business observability artifacts. This guide shows how these prompts fit into the workflow and reduce the complexity at each stage.

The BOS Factory uses GenAI prompts to transform manual work into guided processes. Here's how it works:

üìù Prompt Tips & Tricks
  • Be specific with context. The more detailed information you provide to the prompts, the better the output.
  • Review prompt outputs. GenAI suggestions should be validated by the appropriate role before proceeding.
  • Iterate if needed. It's normal to run prompts multiple times with refinements.
  • Save all artifacts. Templates, reference cards, and dashboard requirements should be stored together.
